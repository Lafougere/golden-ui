Golden UI

An RPG bootstrap compatible theme


Golden UI is a pixel art fantasy RPG user interface designed by Michele "Buch" Bucelli. It has been adapted into a Twitter Bootstrap theme, to be used freely in your websites and browser games as long as proper credit is given. It supports most of Bootstrap's features as well as a few other cool ones.
